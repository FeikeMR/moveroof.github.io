Is verkopen en kopen via MoveRoof gratis?

Ja. MoveRoof is een passie project van mij - Feike Leemkuil. Dit is gratis, omdat ik de kosten die ik had tijdens kopen/verkopen absurd vond en het anders wil voor anderen.

Heb ik verkoopervaring nodig om te verkopen via MoveRoof?

Nee, MoveRoof geeft gratis advies en instructies gedurende het verkoopproces. Verkopen = simpel via MoveRoof.

Wat als ik hulp nodig heb tijdens de verkoop?

Als ik toch een onderdeel niet voldoende heb toegelicht ben ik bereikbaar voor hulp.

Komt mijn woning op Funda?

Nee, in deze eerste versie van MoveRoof niet omdat ik het gratis wil laten. Stel dit werkt goed dan werk ik een Funda plaatsing uit. Kosten hiervoor zullen de kosten zijn die wij hiervoor bij Funda hebben, niks meer.

Hoe snel kan ik mijn woning verkopen?

Sneller dan bij traditionele makelaars, maar dit blijft altijd afhankelijk van marktvraag, hier kunnen wij samen naast goede marketing niet veel aan veranderen.

Kan ik mijn woningpagina elk moment verwijderen of aanpassen?

Ja. Echter, omdat MoveRoof geen data verwerkt zal dit verwijderen/aanpassen in contact met mij moeten. Gelukkig sta ik altijd voor je klaar.

Hoe kan MoveRoof gratis zijn?

Ik (Feike Leemkuil) heb een andere baan voor mijn inkomen. Ook heb ik geleerd om te programmeren en doe ik al het nodige werk zelf dus ik heb geen kosten.
